# vue_tesseract

## 功能描述
- 一个在线ocr的网站，点击[这里](https://zjyfdu.github.io/vue_tesseract/)试用
- 识别用的是现成的tesseract.js，识别效果有点感人
- 好像需要科学上网，而且第一次用的话，要等比较久

## 技术栈
- 代码在code分支
  - 本地server `npm run server`
  - 部署 `npm run build`
- vue + element-ui + vue-cropper + tesseract.js
